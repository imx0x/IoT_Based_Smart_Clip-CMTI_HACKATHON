# IoT_Based_Smart_Clip-CMTI_HACKATHON

## Developed a unique paper clip that can be controlled by a mobile app, gives the exact location of the document it was pinned to, and includes a buzzer for easy location finding.
### Selected as a finalist in the CMTI Hackathon for its innovative use of IoT technology.
